## Overview
This project implements a lexical and syntax analyzer for a C-inspired programming language using Flex and Bison.

The analyzer processes source code, validates its grammar, and detects syntax errors.

## Features
- Lexical analysis with Flex
- Syntax analysis with Bison
- Custom grammar definition for a C-like language
- Error detection and reporting
- Test cases for validation
